Les modules sont de différentes sortes :
- spaceship.modules.Battery
- spaceship.modules.Canon
- spaceship.modules.Command
- spaceship.modules.Engine
- spaceship.modules.Laser
- spaceship.modules.Radar
- spaceship.modules.Room
- spaceship.modules.Shield
- spaceship.modules.SolarPanels
- spaceship.modules.Storage
- spaceship.modules.VSL

Il existe une classe abstraite spaceship.Module qui gère la partie commune à chaque modules.

spaceship.Module

Variables :
int id = identifiant du module dans le vaisseau
Spaceship myParent = accéder au vaisseau parent (= on est le module de quoi ?)

vect3D position = position en km relativement au centre (module de commande) du vaisseau
vect3D positionRelativeBarycentre = position relative au centre de rotation du vaisseau (centre de masse)
vect3D orientation = orientation extrinseque du vaisseau
vect3D AbsolutePosition = position du vaisseau selon le repère universel dans la zone actuelle (voir zones)

float rayon = rayon du module (tout les modules sont des "sphères" dans le moteur)
float poids = masse du module, utile dans le calcul du barycentre

String type = type de module pour le debugage

boolean ferme = sas ouverts ou fermés (true = fermé)
boolean alarme = alarme active ou non
boolean breach = ouverture dans le module ou non
boolean incendie = incendie dans le vaisseau ou non



spaceship.modules.Battery

- Options (fichier story) :
nbHumains possible,
capacité electrique

- Variables :
float capacity
float value

Ce module permet de stocker l'energie électrique. Il peut être rempli grace aux panneaux solaires !

spaceship.modules.Engine

- Options (fichier story) :
mode (0=actionner un instant la force maximum puis stope, 1=actioner ou arreter sont les seules actions, 2=on peut demander un pourcentage de la force maximum),
force maximum (double 1 unité de force = 1km/s^2),
utilisation de l'energie electrique (par unité de force, en kW),
utilisation de carburant (par unité de force, en kW)

- Variables :
float state (allumé ou non)
int choosedMode (permet de changer de mode, exemple passer au mode 0 pour un moment, puis revenir au 2)
int mode (mode par defaut, permet d'empecher un changement de mode vers 2 si initialement le moteur est en 0)
double maxForce
double batteryUse
double carburantUse

Ce module permet de faire bouger le vaisseau, en jouant sur les paramètres, on peut créer différents types de moteurs :
Moteurs de réglage d'orientation faibles et électriques
Boosters à carburant sans réglage de puissance
Moteur secondaires avec réglage de la puissance
...

spaceship.modules.Storage

- Options (fichier story) :
nbHumains possible,
capacité,
type de stockage (0 = carburant, 1 = missiles)

- Variables :
int storageType
float capacity
float value

Ce module, comme le module batterie permet de stocker un élément du jeu.
