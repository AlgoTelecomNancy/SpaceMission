Liste des requetes sur le serveur

Le serveur est accessible en connexion socket sur le port 18000

Pour envoyer des requetes au serveur, il faut d'abord se connecter :
Création des utilisateurs
Administrateur : create admin -p [MOT DE PASSE ADMIN DONNÉ DANS LA CONSOLE JAVA]
Joueur : create player -p [MOT DE PASSE ADMIN DONNÉ DANS LA CONSOLE JAVA]
Un utilisateur est automatiquement reconnecté par lecture de l'adresse ip si déconnecté.
Connexion
Si un joueur a changé d'adresse ip / de machine la connexion se fait de cette façon :
connect -i [numéro de joueur] -p [mot de passe]
connect -n [pseudo de joueur] -p [mot de passe]

Données de jeu
Tout le monde :
players liste les joueurs
Connecté en tant qu'admin :
mainpassword [new password] permet d'afficher le mot de passe de jeu / de le changer
